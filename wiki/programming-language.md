## What is a programming language

A programming language is a formal set of instructions that allows humans to communicate with computers. Programmers write [source code](./software-types.md#source-code) in a programming language, and the computer translates it into machine instructions it can execute.

### `Python`

`Python` is a general-purpose programming language known for its readable syntax and large standard library. It is widely used for [web APIs](./web-api.md#what-is-a-web-api), data science, automation, and scripting.

### `JavaScript`

`JavaScript` is a programming language that runs in the browser and makes web pages interactive. It can update content, respond to events, and communicate with servers.

### `SQL`

`SQL` (Structured Query Language) is a language for managing and querying relational [databases](./database.md#what-is-a-database). Unlike general-purpose languages, `SQL` is designed specifically for storing, retrieving, and manipulating structured data.

### `Rust`

`Rust` is a systems programming language focused on performance, memory safety, and concurrency. It prevents common bugs like null pointer dereferences and data races at compile time.

### `Go`

`Go` is a statically typed programming language developed by Google. It is designed for building scalable network services and [command-line tools](./cli.md#what-is-a-cli), with built-in support for concurrency.

### `TypeScript`

`TypeScript` is a superset of [`JavaScript`](#javascript) that adds static type checking. It compiles to `JavaScript` and runs wherever `JavaScript` runs, including browsers and [`Node.js`](./nodejs.md#what-is-nodejs).

### `Haskell`

`Haskell` is a purely functional programming language with strong static typing. It emphasizes immutability and mathematical function composition, making it popular in academia and for applications requiring high correctness guarantees.

### `Bash`

`Bash` is a [shell](./shell.md#what-is-a-shell) scripting language used to automate tasks and interact with the [operating system](./operating-system.md#what-is-an-operating-system) from the [command line](./cli.md#what-is-a-cli).

### `OCaml`

`OCaml` is a functional programming language with strong static typing and type inference. It supports both functional and imperative programming styles and is used for building compilers, static analysis tools, and financial systems.

### `Nix`

`Nix` is a functional language used to define reproducible software builds and system configurations. It is the language behind the [`Nix` package manager](./nix.md#what-is-nix) and `NixOS`.
